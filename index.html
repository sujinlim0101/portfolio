새로운 지식 습득을 즐기고 변화에 빠른 적응이 가능한 프로그래머입니다.

능동적으로 일하는 개발자 임수진입니다. 그리고 사람에 가치를 두고, 다른 팀과 협업하기를 좋아합니다.
Vue3의 구조를 최대한 활용하여 효율적인 컴포넌트를 만드는 부분을 관심있게 보고 있습니다.
조금 더 나은 코드를 만드는데 관심이 있어 디자인 패턴, 클린 아키텍쳐, 리펙토링 등의 사내 스터디등에도 적극 참여했습니다.
복잡한 내용을 정리하고, 작업을 단순/반복적인 작업을 개선하면서 가이드라인등을 작성하여 팀 효율을 높이는 것에 보람을 느끼는 개발자입니다.

Main Skills

Vue Typescript AngularJS React Jest Sass REST GraphQL
